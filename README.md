# 🎬 Movie IMDB Score Prediction

## 🚀 Project Overview

This project predicts IMDb scores using machine learning.

## 🧠 What I Did

* Data cleaning (handled missing values)
* Feature engineering (log transform, encoding)
* Train-test split
* Linear Regression model
* Model evaluation (Residual plot, Actual vs Predicted)

## 📊 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## 📈 Results

* Evaluated using MSE and R² score
* Visualized model performance using plots

## Take aways

* learned that features matters the most
